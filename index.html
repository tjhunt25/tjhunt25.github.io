About me

As a previous Digital Marketer I have been exposed to many different industries. Being exposed to Web development I have been able to gauge and understand what potential lies underneath the web development industry. The concept of being able to impact society as well as to be able to bring my concepts and ideas to life sounded exhilarating.

This is where I decided to take a 12 week web development immersive course at General Assembly. The course has given me the skills & tools necessary to be able to build secure working full stack applications.

Skills achieved throughout the course:
-Writing web pages in HTML, CSS & JS. Writing web pages in HTML, CSS & JS.
- using MVC frameworks such a Ruby on Rails to build back-end applications.
- Building back-end applications using MVC frameworks (Ruby on Rails).
- developing applications & deploying with cloud services with Heroku.
- Employing JS and Ruby libraries such as Underscore, jQuery and Sinatra.
- Working with APIs
- Version control with github.
